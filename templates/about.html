GeminAI

Horoscope Generation with Machine Learning

This project began after learning about various text generation practices in machine learning. Noticing a pattern of how horoscopes tend to be written by many different websites, we thought it would be a fun project to see if a machine could mimic those patterns, and create its own horoscopes.

With help from the textgenrnn created by Max Woolf, we have compiled a dataset of horoscopes of each sign from 2013 to 2019, and trained a Recurrent Neural Network to produce new horoscopes.
